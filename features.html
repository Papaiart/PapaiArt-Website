Features

Everything a modern animation studio needs — in one focused package

PapaiArt Animation Studio covers nine major capability areas, from low-level geometry editing to final MP4 export.

▢

3D Modeling

Polygonal mesh editing, topology operations, primitive generators. Interactive Bevel and Knife tools.

✎

2D Drawing & Animation

Dedicated vector 2D engine in 3D space. Frame-by-frame animation, drawing layers, onion skinning.

⌨

Animation System

Keyframe-based parameter animation, Graph Editor (F-curves), multi-track timeline.

⚙

Rigging & Constraints

IK, TrackTo, Copy, ChildOf, Limit constraints. For complex skeletal animation.

⊞

Modifiers

Mirror, Subdivision Surface (Catmull-Clark). Non-destructive workflow.

▦

UV & Textures

Interactive UV editor, coordinate manipulation. Image loading and texture mapping.

✦

Rendering & Display

Real-time rendering engine. Custom shader and material system, viewport overlays.

▶

Export & Media

3D mesh import. Direct H.264 MP4 encoding or PNG image sequences.

▤

UI & System

Dockable panel system, infinite Undo/Redo, entity-component scenes, project save/load.

Functionality by category

The full feature set in a single table — by category.

CategoryDetailed functionality
3D Modeling
  • Polygonal mesh editing and topological operations
  • Primitive generation
  • Interactive modeling tools: edge bevel and surface knife
2D Drawing & Animation
  • Dedicated 2D drawing engine and brush tools in 3D space
  • Frame-by-frame 2D animation timeline
  • 2D layer-management system (Drawing Layers Panel)
Animation System
  • Keyframe-based parameter animation
  • Graph Editor for fine-tuning motion
  • Composite timeline view and animation tracks
Rigging & Constraints
  • Inverse Kinematics (IK) for skeletal animation
  • TrackTo and Copy constraints
  • ChildOf hierarchy and Limit constraints
Modifiers
  • Non-destructive editing workflow
  • Real-time symmetry/mirror (Mirror Modifier)
  • Surface subdivision and smoothing
Texturing & UV
  • Interactive UV editor and coordinate manipulation
  • Image loading and texture mapping
Rendering
  • Real-time rendering engine
  • Custom shader and material system
  • Viewport overlays: grid, axes, selection outlines
Export / Import & Media
  • 3D model import (MeshLoader)
  • Direct video encoding to MP4 (H.264)
  • PNG image-sequence output
System & UI
  • Modular, dockable panel system
  • Infinite Undo/Redo history
  • Entity-component scene management and project save/load
