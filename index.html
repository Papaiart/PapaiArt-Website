PapaiArt Animation Studio — Hybrid 2D/3D Animation Software

A single tool for 2D and 3D animation. Draw in 3D space, model non-destructively, rig with inverse kinematics, and render straight to MP4.

High performance  ·  low memory footprint  ·  instant startup
2D drawing

Drawing in real 3D space

2D drawings sit inside the 3D viewport, with cameras, depth and lighting. Vector strokes stay editable, and the brushes (Pencil, Pen, Calligraphy, Rough) respond to tilt and pressure.

  • Drawing tracks and clips (exposure-sheet logic)
  • Onion skinning with customisable ink colours
  • Independent stroke and fill, for flat-design styles
  • Drawing Layers panel: lines, shapes and images as separate sub-layers
3D modeling

Non-destructive 3D workflow

A clean hierarchy panel and a real-time modifier stack. Mirror and Subdivision Surface refresh instantly. Bevel and Knife work interactively on the mesh.

  • Polygon mesh editing & topology operations
  • Primitive generators
  • Custom shader and material system
  • Viewport overlays: grid, axes, selection outlines
Render & Export

From timeline to MP4 in one click

Real-time viewport renderer with configurable resolutions. Export image sequences (PNG) or H.264 MP4 directly. No external editor needed.

  • Built-in VideoEncoder with MP4 (H.264) support
  • Image Sequence output (PNG)
  • FHD presets and custom resolutions
  • Render Animation / Render Single Frame at one click
Why this software?

One tool for both 2D and 3D

Most industry-standard tools specialise in either 2D or 3D. PapaiArt Animation Studio puts both into one workflow.

2D

Drawing in 3D space

A 2D vector engine inside the 3D viewport. Frame-by-frame animation, onion skinning, and vector brushes (Grease Pencil-style).

3D

Non-destructive modeling

Polygonal mesh editing, Bevel and Knife tools, Mirror and Subdivision Surface modifiers in real time.

IK

Rigging and constraints

Inverse Kinematics, TrackTo, ChildOf and Limit constraints. Enough to handle complex character animation.

▶

Built-in MP4 export

Native H.264 video encoder. Render the timeline to a shareable video in one click.

⚡

Quick startup, low memory use

The engine was written for real-time use. The program starts in seconds and stays responsive on modest hardware.

⎘

Modular interface

Dockable panels (Inspector, Viewport, Timeline, Hierarchy). Entity-component scene management, infinite Undo/Redo.

Showcase

What can you create with it?

Characters, hybrid 2D/3D shorts, concept art. The toolset is built around 2.5D visual styles.
